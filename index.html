FancyProtoBox

FancyProtoBox is a fancy, flexible, cool "lightbox" written using Prototype and heavily influenced by FancyBox.

Download FancyProtoBox

Really? Why Another One?!

Well, while looking around for lightboxes I found a few that I liked. There are some very good ones out there and my favorite one was FancyBox. Unfortunately for me, I needed one that was written in Prototype.

So I decided to port FancyBox to Prototype.

How it Works

Basically, you have to remember to include the following files into your project:

  1. Prototype and Scriptaculous
  2. fancyprotobox.js
  3. fancyprotobox.css
  4. All the images found on the images/fancyprotobox folder

There are multiple ways of initalizing it:

You can call the init function and pass a CSS selector to initialize multiple elements at once:

FancyProtoBox.init("div.photo a");

You can create a new FancyProtoBox and pass the id of the link element you want to initialize:

new FancyProtoBox('myFancyLink',{ width:600, height:300 });

You can also pass the id of a hidden div element and then manually show the modal window:

var fancy = new FancyProtoBox('myHiddenDiv',{ width:600, height:300 });
fancy.show();
